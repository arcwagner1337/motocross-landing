# Moto Workshop Landing Page — Сайт-визитка кроссовой мотомастерской

Современный, адаптивный и стильный лендинг для мастерской по ремонту, обслуживанию и кастомизации кроссовых мотоциклов и питбайков. Проект разработан с целью демонстрации навыков качественной контентной верстки, построения удобной навигации и создания привлекательного UI для коммерческой ниши.

## 🛠 Стек технологий

*   **Frontend:** TypeScript, React, Vite
*   **Styling:** Tailwind CSS (полная адаптивность)
*   **Infrastructure:** Docker (быстрое развертывание без внешнего веб-сервера)

---

## 📋 Структура и функционал сайта

Лендинг разбит на логические разделы, полностью закрывающие потребности реального бизнеса:

*   **Главная страница (Main):** Вводная секция с сильным оффером, атмосферой мастерской и кнопками призыва к действию.
*   **Мастерская (Workshop):** Интерактивный блок с прозрачными прайс-листами на услуги (ТО, ремонт двигателей, настройка подвески и т.д.).
*   **Наш гараж (Garage):** Галерея успешно завершенных проектов по кастомизации и восстановлению кроссовой техники.
*   **Команда и Ивенты (Team & Events):** Секция с представлением мастеров и карточками предстоящих или прошедших мото-мероприятий, соревнований и выездов.

## ⚙️ Установка и локальный запуск

Проект максимально автономен. Весь фронтенд упакован в Docker-контейнер, поэтому для запуска не нужно иметь установленный Node.js в системе или настраивать файлы окружения `.env`.

### 1. Клонирование репозитория
```bash
git clone https://github.com/arcwagner1337/motocross-landing
cd motocross-landing
```

### 2. Запуск через Docker Compose
Выполните стандартную команду сборки из корневой папки:
```bash
docker-compose up -d --build
```

После успешного развертывания контейнера, статический сайт будет доступен локально по адресу `http://localhost`.
